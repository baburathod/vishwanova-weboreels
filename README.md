## ✨ Features

- **🎯 Single-Page** — Each weboreel is one self-contained `index.html`
- **📱 Mobile-Friendly** — Fully responsive on all devices
- **🔊 Background Audio** — Web Audio API synthesized music in every project
- **🎨 Unique Visuals** — Canvas animations, particles, gradients, glassmorphism
- **⚡ Zero Dependencies** — Pure HTML + CSS + JavaScript, no frameworks
- **💾 LocalStorage** — Persistent data where applicable (budget, todos, votes)

## 🛠️ Run Locally

```bash
# No install needed — just serve the static files
npx serve -l 3000

# Open in browser
http://localhost:3000
```

## 🏗️ Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, animations, glassmorphism, gradients |
| Vanilla JS | Logic, interactivity, game engines |
| Canvas API | Particle effects, physics, generative art |
| Web Audio API | Synthesized background music & sound effects |
| LocalStorage | Persistent user data |
